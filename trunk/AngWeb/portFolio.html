LRP


Long Range Planning is a web application built for long range resource planning for Abbott R&D.

LRP helps to manage resources and expenditure for various organizations within Abbott R&D. It is port over of legacy LRP Spreadsheets using responsive UI design using Bootstrap, Angular JS, Microsoft Web API 2 and Microsoft SQL Server. The primary goal of this application is to make data entry simple and ready to make business critical insights possible.


Goal


Our goal is to stream line long range planning needs for Abbott R&D, we believe in automated processes to reduce human errors, prevent rework and increase ROI. Most importantly, we are proud to be part of Abbott R&D, it's flavor, it's culture and much more...
